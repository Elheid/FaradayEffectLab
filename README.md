| **Название сущности**               | **Назначение / роль в лаборатории**                          | **Основные параметры**                                  | **Поведение / логика**                                     | **Взаимодействие с пользователем**                                         |
| ----------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------------- |
| **Генератор СВЧ сигнала**           | Источник высокочастотных колебаний, подаёт волну в установку | Частота (по варианту), мощность, уровень ослабления     | Генерирует модульированный сигнал на выбранной частоте     | Пользователь включает/выключает генератор, выставляет частоту и ослабление |
| **Коаксиально-волноводный переход** | Передаёт сигнал от генератора в волновод                     | —                                                       | Преобразует коаксиальный сигнал в волноводный              | Не взаимодействует напрямую — статический объект                           |
| **Прямоугольный волновод**          | Транспортирует волну Н10 к круглому волноводу                | Тип волны: TE10                                         | Направляет волну с минимальными потерями                   | Нет прямого взаимодействия                                                 |
| **Переход к круглому волноводу**    | Преобразует волну Н10 → Н11                                  | —                                                       | Конвертирует тип волны                                     | Статический объект                                                         |
| **Круглый волновод**                | Основная зона взаимодействия феррита и волны                 | Диаметр 2a = 21 мм, тип волны H11                       | Распространяет волну, на которую влияет феррит             | Нет взаимодействия                                                         |
| **Ферритовый стержень**             | Объект, создающий эффект Фарадея                             | Радиус b, длина l, намагниченность M₀                   | Вызывает поворот плоскости поляризации в зависимости от H₀ | Не двигается, но логически реагирует на изменение тока                     |
| **Катушка с током**                 | Создаёт постоянное магнитное поле H₀ вокруг стержня          | Ток I (0–0.6 A), L = 140 мм, D = 30 мм, n = 1500 витков | Меняет величину H₀ = nI / √(L²+D²)                         | Пользователь регулирует ток (ползунок, ввод значения)                      |
| **Конический рупор (излучатель)**   | Излучает волну после прохождения волновода                   | —                                                       | Формирует направленную линейно поляризованную волну        | Нет прямого взаимодействия                                                 |
| **Прямоугольный рупор (приемник)**  | Принимает волну после поворота поляризации                   | Угол поворота                                           | Принимает сигнал ∝ cos(φ)                                  | Пользователь вращает рупор вручную или кнопками                            |
| **Поворотный механизм рупора**      | Позволяет измерять минимум/максимум сигнала                  | Угол φ (0–360°)                                         | Точно вращает рупор; влияет на принимаемый сигнал          | Пользователь вращает рупор с шагом, ищет минимум                           |
| **Измерительный усилитель**         | Показывает уровень низкочастотной огибающей сигнала          | Режим ×1 / ×10, шкала                                   | Отображает амплитуду сигнала после детектора               | Пользователь считывает значения, переключает ×1/×10                        |
| **Блок питания катушки**            | Источник постоянного тока для катушки                        | 0–0.6 A, направление                                    | Задаёт величину H₀                                         | Пользователь управляет током через UI                                      |
